## Download e Instalação:<br><br>Automático (Recomendado)
1. Instale o [DIVA Mod Manager (DMM)](https://github.com/TekkaGB/DivaModManager).
2. Rode o DIVA Mod Manager e faça o Setup.
3. No [GameBanana](https://gamebanana.com/mods/503640), aperte em "Diva Mod Manager 1-CLICK INSTALL".
4. Após isso o mod estará instalado.
## Manual
1. Instale o [DivaModLoader (DML)](https://github.com/blueskythlikesclouds/DivaModLoader) em seu jogo.
2. Baixe o mod pelo [Releases](https://github.com/Heruzinyo/MegaMixPlusPTBR/releases) ou pela página oficial no [GameBanana](https://gamebanana.com/mods/503640).
3. Após ter instalado o DivaModLoader (DML) e extraído o .7z do mod, coloque a pasta "MegaMix+PTBR" em "mods" no diretório do jogo.
4. Após isso o mod estará instalado.
